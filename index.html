Calculadora de Sistemas de Colas

Bienvenido a la calculadora de sistemas de colas. Esta herramienta te permite calcular métricas importantes para diferentes modelos de teoría de colas.

Selecciona un modelo del menú lateral para comenzar a calcular las métricas de rendimiento de tu sistema de colas.

Modelos Disponibles

M/M/1

Sistema con llegadas Poisson, servicio exponencial y un servidor.

M/M/2

Sistema con llegadas Poisson, servicio exponencial y dos servidores.

M/M/1/N

Sistema M/M/1 con capacidad máxima limitada a N clientes.

M/G/1

Sistema con llegadas Poisson, servicio general y un servidor.

M/D/1

Sistema con llegadas Poisson, servicio determinístico y un servidor.

Prioridades

Sistema M/M/1 con dos clases de prioridad no expulsiva.

Modelo M/M/1

El modelo M/M/1 representa un sistema de colas con llegadas que siguen un proceso de Poisson, tiempos de servicio exponenciales y un solo servidor. Es el modelo más básico y fundamental en la teoría de colas.
